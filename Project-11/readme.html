My Spotify Clone

Overview

Welcome to the My Spotify Clone project! This is a web application that mimics some functionalities of Spotify, allowing users to browse and play music playlists. The project is built using HTML, CSS, JavaScript, and includes a simple API to handle music data.

Project Structure

  • View Live: My Spotify Clone
  • View Source Code: GitHub Repository

Contents

  1. Header
    • Displays the application logo and a navigation menu.
  2. Playlists
    • Users can browse different playlists, each containing a set of songs.
  3. Player
    • Music player that allows users to play, pause, and skip songs.
  4. Styling with CSS
    • Utilizes CSS Flexbox and Grid for layout and positioning.
    • Implements a responsive design with media queries.

How to Use

  1. Clone the repository: git clone https://github.com/RahulP-Here/my-spotify-clone.git
